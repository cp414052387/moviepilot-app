# 📌 核心主角

## 陈平安

| 属性 | 详情 |
|------|------|
| **身份** | 小说男主角、落魄山山主、剑气长城末代隐官 |
| **出身** | 骊珠洞天小镇泥瓶巷，自幼父母双亡，家境贫寒 |
| **师承** | 儒家文圣关门弟子 |
| **修为** | 止境归真圆满武夫，仙人境剑修 |
| **本命飞剑** | 笼中雀、井口月、青萍、北斗（四把） |
| **性格** | 坚韧、善良正直，坚守赤子之心，走圣人之道 |
| **感情** | 宁姚未婚夫 |

### 人物生平
1. **幼年**：生于骊珠洞天泥瓶巷，5岁时本命瓷被碎，父母双亡
2. **少年**：依靠在瓷窑劳作和替邻居担水送信维持生计
3. **转折**：齐静春离世后，为护送李宝瓶赴山崖书院，踏上修行之路
4. **游历**：远赴大隋，结识阿良、魏檗、文圣等重要人物
5. **立山**：购买落魄山，建立宗门
6. **成就**：合道半座剑气长城，成为仙人境剑修、剑气长城末代隐官

# 🏛️ 三教祖师

## 概述

三教祖师是人间仅有的三位十五境大佬，代表了道教、儒教和佛教的巅峰实力。

| 称号 | 教派 | 修为 | 特点 |
|------|------|------|------|
| **道祖** | 道教 | 十五境巅峰 | 公认战力最强，最会打架，青冥天下之主 |
| **至圣先师** | 儒教 | 十五境 | 原型孔子，以读书入道，炼化万卷为本命物 |
| **佛祖** | 佛教 | 十五境 | 莲花天下最高代表，兼具佛法与武道双重修行 |

## 散道

三教祖师因被周密自爆波及，最终选择散道，共同抵御蛮荒天道。
